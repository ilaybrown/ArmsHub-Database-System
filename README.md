# Database Design & SQL Project

This project presents a full design and implementation of a relational database, including modeling, schema creation, data generation, and custom SQL queries.

## Work Summary

• Built a complete business story defining entities and relationships.  
• Designed a structured ERD based on the story.  
• Created the full DDL schema with appropriate constraints.  
• Generated and loaded dataset matching the schema.  
• Designed original SQL queries based on the model.  
• Implemented query solutions using JOINs, aggregations, filters, and views.  
• Validated results and refined data to ensure correctness and non-empty outputs.

## Files Included

• Story description  
• ERD diagram  
• DDL schema  
• Dataset  
• SQL queries  
• Query solutions
